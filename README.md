## Meu Portfólio - Desenvolvedor

Um site de portfólio profissional feito com ASP.NET Core MVC. Mostro meus projetos e habilidades como desenvolvedor, com um formulário de contato totalmente funcional.

## O que tem no projeto
Página inicial com minhas informações
Links diretos para meus contatos (email, GitHub, LinkedIn)
Código organizado seguindo padrões MVC

## Como rodar na sua máquina
O que você precisa ter instalado:
.NET 8.0 SDK
Visual Studio 2026 
Git

Passos:
Baixe o projeto:
bash
git clone https://github.com/MicaelyCacho/MeuPortifolio.git

Entre na pasta:
bash
cd MeuPortifolio

Restaure as dependências:
bash
dotnet restore

Rode o projeto:
bash
dotnet run

Abra no navegador:
http://localhost:5000
https://localhost:5001

Ou acesse o link de deploy: https://meuportifolio.fly.dev/
